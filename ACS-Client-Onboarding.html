Client Onboarding

Manage Client

To manage the client, go to Network Setup > Manage Client, as shown below:

Now, click on the Manage Client Onboarding, the manage client onboarding page will come on to the screen as shown below:

On Manage Client Page, you can see the details of all the clients i.e., Account ID, Company Name, Contact Name, Contact Email, Address, Status and Action.

On Manage Client page, you can add a new client or filter manage client page with the help of filter tab.

Add a New Client

To add a new client, click on the Add Client button as shown below, the add client page will appear.

Now fill the details Client Info, Billed By, LOB Configuration, Product Categories, Client Settings, Client FC Configuration (If you want to add more FC then click on the Add More Button).

Now click on the Submit Button to add the client.

Filter the Manage Client Page

If you want to filter this page, then click on the Filter the list of filters will be shown as below:

Now fill the details, with the help of Drop and Down Button of the Account Id, Company Name and Status and then click on the search button, the filter client list will be come on the screen.

If you want to delete the filter data then click on the Reset button, the filter data will be deleted.

Manage Client Facility

To manage the client facility, go to Network Setup > Manage Client Onboarding, as shown below:

Now, click on the Manage Client Facility, the manage client facility page will come on to the screen as shown below:

On Manage Client Facility Page, you can see the details of all the facility i.e. Facility Code, Facility Name, Account ID, Contact Name, Contact Email, Address, Status and Action.

On Manage Client page, you can add a new facility or filter the manage client facility details with the help of filter tab.

Add a New Facility

To add a new facility, click on the Add Facility button as shown below, the add facility page will appear.

Now fill the details Client Facility, Operating Hours, Holiday (If Applicable), WSE Carrier Network Applicability (If you want to add more Carrier Type then click on the Add More Button).

Now click on the Submit Button to add the client facility.

Filter the Manage Client Page

If you want to filter this page, then click on the Filter the list of filters will be shown as below:

Now fill the details, with the help of Drop and Down Button of the Account ID, Facility Code, Facility Name and Status and then click on the search button, the filter client list will be come on the screen.

If you want to delete the filter data then click on the Reset button, the filter data will be deleted.
